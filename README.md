# SeleniumTestNGFramework

A clean and scalable Selenium automation framework built with **TestNG** and **Maven**, providing a glimpse into building maintainable QA automation from scratch.


## Tech Stack

- **Selenium WebDriver** – Browser automation
- **TestNG** – Test framework and lifecycle management
- **Maven** – Build and dependency management
- **Allure Reporting** – Test reporting with screenshots for pass/fail steps
- **Java 25** – Programming language

---

## Key Features

- **Page Object Model (POM)** for clean and maintainable test design  
- **Reusable Utils Layer** for actions, waits, scrolling, and other helpers  
- **TestNG Lifecycle Management** with proper setup and teardown  
- **Allure Reporting** with clear steps and screenshot attachments  
- **Meaningful Assertions** for reliable test results
